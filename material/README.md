# material/ — dein Quellmaterial (lokal, nie im Repo)

Lege hier die Schulungs-PDFs/PowerPoint-Exporte/Notizen ab, aus denen die Fragen
entstehen sollen. **Dieser Ordner ist per .gitignore vollständig ausgeschlossen** –
nichts davon landet im öffentlichen Repository.

Ablauf:
1. Dateien hier ablegen (oder direkt im Chat anhängen).
2. Claude wandelt sie – streng quellengebunden – ins Frageformat um.
3. Prüf-Liste „Frage → richtige Antwort → Quellbeleg (Folie X)" zum Freigeben.
4. Freigegebene Inhalte gehen **gegated in Supabase** (nicht ins Repo).
